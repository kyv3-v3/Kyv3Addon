# Kyv3 Flow Addon

Now u can make modules without coding.

## Modules
- `flow-forge`: visual editor for custom modules.
- `flow-module-manager`: manager for custom modules.


## Requirements
- Java 21
- Minecraft `1.21.11`
- Meteor Client `1.21.11`

## Build and run
1. Build the project:
   `./gradlew build`
2. Run tests:
   `./gradlew test`
